# LLM (Large Language Model): Transformer / BERT / GPT

This is a project to implement large language models, including BERT, GPT, and Transformer. The project is based on PyTorch.

My Implementation:
- BERT: [link](./model/model_bert.py) This is a base model that can be used for various purposes such as Masked Language Modelling, Classification, or any other kind of NLP tasks.
- GPT: [link](./model/model_gpt.py) This is a base model that can be used for various purposes such as Language Modelling, Text Generation, or any other kind of NLP tasks.
- Transformer: [link](./model/model_transformer.py) This is a base model that can be used for various purposes such as Language Modelling, Text Translation, or any other kind of NLP tasks.
