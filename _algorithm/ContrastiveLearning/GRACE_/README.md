## GRACE: Deep Graph Contrastive Representation Learning

The official PyTorch implementation of deep **GRA**ph **C**ontrastive r**E**presentation learning (GRACE)

### Usage

```bash
# `--dataset` argument should be one of [ Cora, CiteSeer, PubMed, DBLP ]
$ python main.py --dataset Cora  
```
