# DeepInsight & scDeepInsight

DeepInsight is a methodology to transform a non-image data to an image for convolution neural network architecture, but can be applied to other types of high-dimensional data as well. scDeepInsight is an extension of DeepInsight that is specifically designed for single-cell RNA-seq data.
