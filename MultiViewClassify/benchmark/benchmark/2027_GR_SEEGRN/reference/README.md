# Comparison Method


## 1. Method

### Supervised Learning: 

#### Graph Neural Network Based

- GENELink: Graph attention network for link prediction of gene regulations from single cell RNA-sequencing data

    - PAPER: [Graph attention network for link prediction of gene regulations from single cell RNA-sequencing data](https://academic.oup.com/bioinformatics/article/38/19/4522/6663989)
    - GITHUB: [GENELink](https://github.com/zpliulab/GENELink)

- DeepTFni: A VGAE-based model to infer transcription factor regulatory network

    - PAPER: [Inferring transcription factor regulatory networks from single-cell ATAC-seq data based on graph neural networks](https://www.nature.com/articles/s42256-022-00469-5)
    - GITHUB: [DeepTFni](https://github.com/sunyolo/DeepTFni)


#### Non-Graph Neural Network Based

- CNNC: Deep learning for inferring gene relationships from single-cell expression data

    - PAPER: [Deep learning for inferring gene relationships from single-cell expression data](https://www.pnas.org/doi/10.1073/pnas.1911536116)
    - GITHUB: [CNNC](https://github.com/xiaoyeye/CNNC)

- DGRNS: A deep learning framework for gene regulatory network inference from single-cell transcriptomic data

    - PAPER: [DGRNS: A deep learning framework for gene regulatory network inference from single-cell transcriptomic data](https://academic.oup.com/bioinformatics/article/39/4/btad165/7099621)
    - GITHUB: [DGRNS](https://github.com/April-Secret/DGRNS)

- STGRNS: an interpretable transformer-based method for inferring gene regulatory networks from single-cell transcriptomic data

    - PAPER: [STGRNS: an interpretable transformer-based method for inferring gene regulatory networks from single-cell transcriptomic data](https://academic.oup.com/bioinformatics/article/39/4/btad165/7099621)
    - GITHUB: [STGRNS](https://github.com/zhanglab-wbgcas/STGRNS)


### Unsupervised Learning

- DeepSEM: Modeling Gene Regulatory Networks Using Neural Network Architectures

    - PAPER: [Modeling Gene Regulatory Networks Using Neural Network Architectures](https://www.nature.com/articles/s43588-021-00099-8)
    - GITHUB: [DeepSEM](https://github.com/HantaoShu/DeepSEM/tree/master)
